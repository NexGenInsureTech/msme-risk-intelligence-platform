# Sprint 3 — MVP Development

## Sprint Goal

Begin development of the MSME Risk Intelligence Platform MVP.

This sprint focuses on building the foundational software components required to run the platform.

---

## Development Strategy

The MVP will be built using a monorepo architecture.

Technology stack:

Frontend  
React (Next.js)

Backend  
Python FastAPI

Database  
PostgreSQL

Risk Engine  
Python modules integrated into backend services

---

## Sprint Tasks

1. Create backend project structure
2. Initialize FastAPI server
3. Implement risk engine module
4. Define database models
5. Create API endpoints for risk analysis
6. Create frontend project structure
7. Build discovery interface

---

## Expected Deliverables

/backend/app/main.py  
/backend/services/risk_engine.py  
/backend/models/msme.py  
/frontend/app/page.tsx  
/frontend/components/risk-form.tsx

---

## Sprint Outcome

At the end of this sprint the platform should:

• run a backend API server  
• accept MSME input data  
• calculate risk scores  
• display risk intelligence results on the frontend
